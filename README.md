# Smart B-Roll Placement Detector

AI-powered tool that detects ideal B-roll placements from video transcripts using FastAPI and React.

## Features
- Detects cinematic moments
- Suggests timestamp placements
- Confidence scoring
- Full-stack architecture
- Responsive UI

## Tech Stack
- React.js
- FastAPI
- Python
- CSS

## Demo
Paste a transcript and receive suggested B-roll insertion timestamps automatically.
